# 安装花生壳

闲置花生壳3个月了，用树莓派可当花生棒

安装方法：
进入下载好的花生壳配置文件放到本地目录
解压phddns2后进入目录:
sudo tar zxvf phddns_raspberry.tgz

运行配置文件：sudo ./oraynewph start
安装启动后获取的sn为绑定代码，到花生壳官网进行绑定登录
默认绑定密码为：admin


状态命令：
启动：oraynewph start
查看状态：oraynewph status
重置应用：oraynewph reset
停止：oraynewph stop
卸载：oraynewph uninstall
